## Funcionalidad

- Este script es un conjunto de herramientas automatizadas que contiene:

- fierce
- sslscan
- nmap
- dmitry
- nikto
- whatweb
- davtest
- unicornscan
- dnsmap

## instalacion

- git clone https://github.com/hackingboy2020/HyperScaner.git
- chmod 777 -R HyperScaner
- cd HyperScaner
- chmod +x *
- bash install.sh
- bash hyperscaner.sh
